Aide – Comment utiliser AcroJuge

1) Configuration
  • Ajoute les observateurs (séparés par virgule, point-virgule, ou retour à la ligne).
  • Ajoute le groupe observé (prénoms) : il servira pour l’évaluation individuelle.
  • Renseigne le plan d’enchaînement : nombre de pyramides à 2 / 3 / 4 / 5 / libres.
  • Appuie sur Démarrer l’observation pour créer l’enchaînement et passer à l’observation.
2) Observation
  • Pour chaque pyramide : Tenue 3s (OUI/NON) et Sécurité pour chaque critère :
  • Codes : 🔴 Fragile, 🟡 Moyen, 🟢 Bon, 🟩 Très bon (vert foncé).
  • Besoin de revoir un autre enchaînement ? Utilise 📂 Enchaînements.
3) Évaluations
  • Évaluation de l’ensemble : choisis un niveau F/M/B/TB pour Composition, Entrée, Sortie, Originalité.
  • Évaluation individuelle : saisis une note /20 par élève (demi-points possibles). Les champs sont vides par défaut.
  • Ajoute un commentaire global et une note finale /20 si souhaité (demi-points possibles).
  • Termine par ➕ Nouvelle observation si un autre groupe doit être évalué (remet à zéro le groupe et le plan).
4) Export
  • Depuis l’en-tête, clique 📄 Exporter TOUT (PDF) : un seul document avec toutes les observations (pyramides détaillées + ensemble + individuel).
  • Le PDF a un pied de page et est conçu pour être lisible et pro (A4 portrait). Airdrop OK.
  • Tout est enregistré localement (iPad). Tu peux quitter l’app et reprendre plus tard.
Conseils iPad
  • Ajouter l’app sur l’écran d’accueil (Safari ⋯ → Sur l’écran d’accueil).
  • Si le PDF ne se télécharge pas, utilise Imprimer → Enregistrer en PDF.
